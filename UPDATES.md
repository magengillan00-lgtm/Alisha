# تحديثات Alisha - Alisha Updates

## 🔧 الإصلاحات الرئيسية | Main Fixes

### 1. إصلاح مشكلة الاتصال بـ Hugging Face API
**المشكلة**: كانت الطلبات ترجع خطأ 404 عند محاولة الاتصال بـ `api-inference.huggingface.co/models/`

**الحل**: تم تحديث الـ endpoint إلى `router.huggingface.co/v1/chat/completions` وهو الـ endpoint الصحيح والحديث الذي توفره Hugging Face حالياً.

```javascript
// قبل (Before):
fetch(`https://api-inference.huggingface.co/models/${SELECTED_MODEL}`)

// بعد (After):
fetch('https://router.huggingface.co/v1/chat/completions')
```

### 2. تشفير مفتاح Hugging Face API
**الميزة**: تم إضافة نظام تشفير بسيط لتخزين المفتاح بشكل آمن

```javascript
// التشفير (Encryption):
const encryptedKey = CryptoUtils.encode(key);
localStorage.setItem('ALISHA_HF_KEY_ENC', encryptedKey);

// فك التشفير (Decryption):
const decryptedKey = CryptoUtils.decode(encryptedKey);
```

## 🎯 كيفية الاستخدام | How to Use

### الخطوة الأولى: إدخال مفتاح API
1. عند فتح الصفحة، ستظهر نافذة "إعدادات Alisha"
2. أدخل مفتاح Hugging Face الخاص بك (يجب أن يبدأ بـ `hf_`)
3. اختر نموذج الذكاء الاصطناعي
4. اختر اللغة
5. اختر الموديل (Avatar)
6. اضغط "حفظ وتفعيل"

### الخطوة الثانية: التفاعل مع Alisha
- **الكتابة**: اكتب رسالة في حقل الإدخال واضغط Enter أو اضغط زر الإرسال (➤)
- **الصوت**: اضغط زر الميكروفون (🎤) لإدخال صوتي
- **تغيير اللغة**: اضغط زر 🌐 لتبديل اللغة
- **الإعدادات**: اضغط زر ⚙️ لتغيير الإعدادات

## 🐛 استكشاف الأخطاء | Troubleshooting

### مشكلة: "خطأ في الاتصال بالإنترنت"
**الحل**:
1. تحقق من اتصالك بالإنترنت
2. تأكد من صحة مفتاح API
3. حاول تحديث الصفحة

### مشكلة: "مفتاح API غير صالح"
**الحل**:
1. تحقق من أن المفتاح يبدأ بـ `hf_`
2. تأكد من نسخ المفتاح بشكل كامل بدون مسافات
3. جرب إنشاء مفتاح جديد من حسابك على Hugging Face

### مشكلة: "النموذج قيد التحميل"
**الحل**:
1. انتظر بضع ثوانٍ
2. حاول مرة أخرى
3. جرب نموذج مختلف (مثل Qwen 2.5)

### مشكلة: "الموديل لا يظهر"
**الحل**:
1. تأكد من أن ملفات الموديل موجودة في المجلد الصحيح
2. حاول تحديث الصفحة
3. افتح أدوات المطور (F12) للتحقق من الأخطاء
