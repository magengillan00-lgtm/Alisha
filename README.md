# 🚀 Alicia - Realtime Avatar AI (v0.3)

![Alicia Avatar](alicia_avatar.png)

أليشيا هي مساعدة ذكية وودودة بهيئة شخصية أنمي، مصممة للتفاعل الصوتي المباشر باستخدام تقنيات الذكاء الاصطناعي المتقدمة. تم تحسين هذا المشروع ليعمل بسلاسة مع واجهة مستخدم تعتمد على نماذج VRM ثلاثية الأبعاد.

## ✨ المميزات الجديدة
- **تكامل Gemini AI المطور**: يقوم النظام الآن بالبحث عن أفضل موديل متاح (مثل Gemini 1.5 Pro أو Flash) لضمان أفضل جودة للردود.
- **إصلاح مشكلة API**: تم تحديث مفتاح API الخاص بـ Google لضمان استجابة الأفاتار للأوامر الصوتية.
- **تنظيم الملفات**: تم ترتيب هيكل المشروع بشكل احترافي لتسهيل الإدارة والتطوير.
- **تحسين Lip-Sync**: مزامنة حركة الفم مع الصوت المولد لزيادة الواقعية.
- **دعم اللغة العربية**: الأفاتار يتحدث ويرد باللغة العربية بأسلوب مرح ومختصر.

## 📂 هيكل المشروع المنظم
تم تنظيم الملفات لتكون مبعثرة أقل وأكثر وضوحاً:
- `/assets/animations/`: يحتوي على ملفات الحركة (FBX) مثل `idle` و `talking`.
- `/assets/models/`: يحتوي على نموذج الأفاتار `waifu.vrm`.
- `/assets/backgrounds/`: يحتوي على خرائط البيئة والإضاءة `background.hdr`.
- `/assets/audio/`: ملفات الصوت المرجعية.
- `/assets/videos/`: ملفات الفيديو التوضيحية.
- `cliente_final.html`: الواجهة الأمامية للمشروع.
- `avatar03.ipynb`: الملف البرمجي الأساسي (Alicia Engine) الذي يدير الذكاء الاصطناعي.

## 🛠️ كيفية التشغيل
1. افتح ملف `avatar03.ipynb` في Google Colab.
2. قم بتشغيل الخلايا (Cells) بالترتيب.
3. سيقوم النظام بتثبيت المكتبات اللازمة وتهيئة نموذج Whisper للتعرف على الكلام.
4. بعد التشغيل، ستحصل على رابط Ngrok، افتحه للوصول إلى واجهة الأفاتار.

## 🧠 التقنيات المستخدمة
- **Three.js & @pixiv/three-vrm**: لعرض وتحريك الأفاتار ثلاثي الأبعاد.
- **Google Gemini API**: للتفكير وتوليد الردود الذكية.
- **Faster-Whisper**: لتحويل الكلام إلى نص بسرعة عالية.
- **gTTS**: لتحويل النص إلى كلام باللغة العربية.
