# 🌸 Alicia AI Avatar - GitHub Edition

![Alicia Avatar](./alicia_avatar.png)

**Alicia** هي مساعدة ذكية ثلاثية الأبعاد تعتمد على تقنيات الذكاء الاصطناعي المتقدمة، مصممة لتعمل مباشرة في المتصفح عبر استضافة **GitHub Pages**.

## ✨ المميزات الجديدة
- **الاستضافة:** تم نقل المشروع بالكامل من Firebase إلى GitHub Pages لسهولة الوصول والإدارة.
- **الأمان:** تمت إضافة واجهة لطلب مفتاح API الخاص بـ OpenRouter عند فتح الموقع، مما يمنع سرقة المفاتيح المخزنة في الكود.
- **الواجهة:** تم توسيط الأفاتار في منتصف الشاشة مع التركيز على الجزء العلوي من الجسم.
- **اللغة:** أليشيا ترد الآن باللغة اليابانية دائماً، حتى لو كانت المدخلات باللغة العربية.
- **التفاعل:** صندوق رسائل لإرسال الأوامر، مع دعم لنطق الردود باليابانية وتحريك الشفاه (Lip Sync).

## 🚀 كيفية التشغيل
1. قم بزيارة رابط المشروع على GitHub Pages (سيتم تفعيله بعد الرفع).
2. عند فتح الموقع لأول مرة، سيطلب منك إدخال مفتاح API الخاص بـ **OpenRouter**.
3. يمكنك الحصول على المفتاح من [OpenRouter](https://openrouter.ai/).
4. ابدأ المحادثة مع أليشيا باللغة العربية أو أي لغة أخرى، وسترد عليك باليابانية.

## 🛠 التقنيات المستخدمة
- **Three.js:** محرك الرسوميات ثلاثية الأبعاد.
- **@pixiv/three-vrm:** لتحميل وتشغيل نموذج VRM.
- **OpenRouter API:** لتوليد الردود الذكية (Gemini 2.0 Flash).
- **Web Speech API:** لنطق الردود باللغة اليابانية.

## 📂 هيكل المشروع
- `index.html`: الملف الرئيسي للواجهة والمنطق.
- `assets/models/waifu.vrm`: نموذج الأفاتار.
- `assets/animations/`: ملفات الحركات (FBX).
- `assets/backgrounds/`: خلفية البيئة (HDR).
