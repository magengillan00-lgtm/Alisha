# 🚀 Alicia AI - Realtime Interactive Avatar (v0.4)

![Alicia Avatar](alicia_avatar.png)

أليشيا هي مساعدة ذكية وودودة بهيئة شخصية أنمي، تعمل الآن **مباشرة في المتصفح** دون الحاجة إلى خوادم خارجية أو Google Colab. تم دمج تقنيات الذكاء الاصطناعي للتعرف على الصوت والتحدث مباشرة عبر واجهة الويب.

## ✨ المميزات الجديدة (تحديث Firebase)
- **التفاعل المباشر في المتصفح**: تم نقل منطق الذكاء الاصطناعي ليعمل داخل المتصفح باستخدام Web Speech API و Gemini API مباشرة.
- **تكامل Gemini API**: تم إصلاح مشكلة مفتاح API ودمجه في كود JavaScript للتفاعل الفوري.
- **التعرف على الكلام (STT)**: الأفاتار يستمع إليك الآن عبر الميكروفون باستخدام تقنيات المتصفح المدمجة.
- **تحويل النص إلى كلام (TTS)**: الأفاتار يرد عليك بصوت طبيعي باللغة العربية مع مزامنة حركة الفم.
- **تنظيم الملفات**: تم ترتيب هيكل المشروع بشكل احترافي لتسهيل الرفع على Firebase Hosting.

## 📂 هيكل المشروع المنظم
- `/assets/animations/`: ملفات الحركة (FBX) مثل `idle` و `anim_1, 2, 3`.
- `/assets/models/`: يحتوي على نموذج الأفاتار `waifu.vrm`.
- `/assets/backgrounds/`: يحتوي على خلفية الإضاءة `background.hdr`.
- `cliente_final.html`: الملف الأساسي للواجهة البرمجية والتفاعل.

## 🛠️ كيفية التشغيل والرفع
1. قم برفع جميع الملفات إلى **Firebase Hosting** أو أي مستضيف ويب يدعم الملفات الثابتة.
2. تأكد من وضع جميع الملفات في مجلد `public` إذا كنت تستخدم Firebase.
3. افتح الرابط في متصفح يدعم الميكروفون (يفضل Chrome).
4. اضغط على أيقونة الميكروفون 🎤 وابدأ التحدث مع أليشيا!

## 🧠 التقنيات المستخدمة
- **Three.js & @pixiv/three-vrm**: لعرض وتحريك الأفاتار ثلاثي الأبعاد.
- **Google Gemini API**: للتفكير وتوليد الردود الذكية (مباشرة من المتصفح).
- **Web Speech API**: للتعرف على الكلام وتحويله إلى نص، ولتوليد الصوت.
