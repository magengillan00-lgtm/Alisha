## ✨ المميزات الرئيسية

<table>
<tr>
<td width="50%">

### 🤖 ذكاء اصطناعي متقدم
- **Groq** — Llama 3.3 70B *(افتراضي، الأسرع)*
- **Gemini 2.0 Flash** — عبر HuggingFace Space
- **KiloClaw Backend** — خادم خاص احتياطي
- تبديل ذكي تلقائي عند انقطاع أي مزود
- اختيار المزود يدوياً من الإعدادات

### 🌐 دعم متعدد اللغات
- 🇸🇦 **العربية** — ردود + صوت عربي
- 🇺🇸 **English** — Full English support
- 🇯🇵 **日本語** — 完全日本語対応

</td>
<td width="50%">

### 🎨 تجربة بصرية غامرة
- أفاتارات Live2D بحركات طبيعية
- أفاتار VRM ثلاثي بتنفس وإيماءات وحركة عيون
- مزامنة شفاه مع الصوت (Lip Sync)
- 3 خلفيات أنمي: فضاء 🌌 · حديقة 🌸 · غرفة 🏠

### 🔊 صوت وتفاعل
- تحويل نص إلى كلام (TTS)
- اختيار الصوت حسب اللغة
- إدخال صوتي بالميكروفون
- واجهة نظيفة: الأفاتار يملأ الشاشة بلا عوائق

</td>
</tr>
